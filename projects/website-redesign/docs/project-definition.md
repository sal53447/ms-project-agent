# Project Definition — Website Redesign

## Overview

The Website Redesign project is a ground-up rebuild of the company's public-facing website. The current site is 6 years old, built on a legacy CMS, and no longer reflects the company's brand or product offering. The goal is to launch a modern, fast, accessible website with a refreshed visual identity, improved information architecture, and a content management system the marketing team can operate without developer help.

## Goals

- Deliver a redesigned website that reflects the updated brand identity approved in Q1 2026.
- Improve Core Web Vitals scores (LCP < 2.5s, CLS < 0.1, FID < 100ms).
- Increase organic search traffic by 20% within 6 months of launch (via improved SEO structure).
- Reduce time for marketing to publish new content from 3 days (developer involvement) to same-day self-service.
- Achieve WCAG 2.1 AA accessibility compliance.

## Success Criteria

- All pages pass Lighthouse performance score >= 85.
- Marketing team signs off on CMS usability after a 1-week trial period.
- Zero P1/P2 bugs at launch.
- GA4 bounce rate drops by at least 10% versus current site within 30 days post-launch.
- Site passes automated accessibility audit with no critical violations.

## Scope — In Scope

- Redesign of all top-level pages: Home, About, Products (all sub-pages), Pricing, Blog, Contact, Careers.
- New design system / component library.
- Migration from legacy CMS to Contentful headless CMS.
- SEO audit and implementation of recommendations (meta tags, sitemap, structured data).
- Analytics instrumentation (GA4 + Hotjar).
- Integration with existing HubSpot lead capture forms.
- Responsive design for desktop, tablet, and mobile.
- Redirect mapping from old URLs to new URLs.

## Scope — Out of Scope

- Customer portal / logged-in experience (separate project).
- E-commerce / checkout flow.
- Mobile app.
- Backend API changes.
- Social media assets (handled by brand team separately).
- Any pages not listed in the agreed page inventory (v1.2, attached in Sharepoint).
